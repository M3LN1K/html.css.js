Подробный разбор кода
Полный код:
php
foreach ($arResult['ITEMS'] as $index => $item){
    global $APPLICATION;
    $currentPage = $APPLICATION->GetCurPage();

    if($item['CODE'] == 'floor' && $currentPage == '/kupit-uchastok/'){
        unset($arResult['ITEMS'][$index]);
    }
}
Построчный разбор:
1. foreach ($arResult['ITEMS'] as $index => $item){
foreach - циклическая конструкция PHP для перебора массивов

$arResult['ITEMS'] - массив элементов (фильтров), который нужно обработать

$arResult - стандартная переменная в компонентах Bitrix, содержит результат работы компонента

['ITEMS'] - ключ массива, содержащий элементы фильтров

as $index => $item - присваивает:

$index - ключ текущего элемента (например, 0, 1, 2...)

$item - значение текущего элемента (массив с данными фильтра)

Пример массива $arResult['ITEMS']:

php
$arResult['ITEMS'] = [
    0 => ['CODE' => 'price', 'NAME' => 'Цена', ...],
    1 => ['CODE' => 'floor', 'NAME' => 'Этажи', ...], 
    2 => ['CODE' => 'area', 'NAME' => 'Площадь', ...]
];
2. global $APPLICATION;
global - ключевое слово PHP для доступа к глобальной переменной из локальной области видимости

$APPLICATION - глобальный объект Bitrix CMS, который содержит:

Информацию о текущей странице

Методы для работы с контентом

Данные о сайте

Эта строка делает переменную $APPLICATION доступной внутри цикла

3. $currentPage = $APPLICATION->GetCurPage();
$APPLICATION->GetCurPage() - метод объекта $APPLICATION

Возвращает текущий URL страницы без домена

Например: /kupit-uchastok/ или /catalog/

$currentPage = - сохраняет результат в переменную $currentPage

Примеры возвращаемых значений:

Главная страница: /

Страница каталога: /catalog/

Страница "купить участок": /kupit-uchastok/

4. if($item['CODE'] == 'floor' && $currentPage == '/kupit-uchastok/'){
Условие с оператором И (&&):

Первая часть: $item['CODE'] == 'floor'

Проверяет, равен ли код текущего элемента 'floor'

$item['CODE'] - получает значение ключа 'CODE' из массива текущего элемента

== - оператор сравнения (равно)

Вторая часть: $currentPage == '/kupit-uchastok/'

Проверяет, равен ли текущий URL '/kupit-uchastok/'

Оператор && (логическое И):

Условие истинно только если ОБА выражения истинны

То есть: если ЭТО фильтр "этажи" И мы на странице "купить-участок"

5. unset($arResult['ITEMS'][$index]);
unset() - функция PHP для удаления переменной или элемента массива

$arResult['ITEMS'][$index] - указывает на конкретный элемент массива:

$arResult['ITEMS'] - исходный массив

[$index] - конкретный ключ (например, 1)

Что делает: удаляет элемент фильтра с кодом 'floor' из массива $arResult['ITEMS']

6. } (закрывающая фигурная скобка)
Закрывает блок условия if

В конце цикла неявно закрывается цикл foreach

Как работает весь код:
Шаг за шагом:
Цикл начинается с первого элемента массива $arResult['ITEMS']

Получает данные о текущей странице через $APPLICATION

Проверяет условие:

Это фильтр "этажи"?

Мы на странице "/kupit-uchastok/"?

Если ДА - удаляет этот фильтр из массива

Переходит к следующему элементу массива

Повторяет для всех элементов
